스토어 추가하기

우리는 이미 스토리지 제공자, 모델 제공자, 임베딩 제공자를 추가했습니다. 이제 이러한 제공자들을 사용하기 위한 스토어를 구성해야 합니다.

주의

이 가이드는 이미 Casibase 지식 시스템을 배포했다고 가정합니다. 아직 배포하지 않았다면 Casdoor와 Casibase 배포하기 튜토리얼을 참조하세요.

또한, 이 가이드는 이미 스토리지 제공자, 모델 제공자, 임베딩 제공자를 추가했다고 가정합니다. 아직 추가하지 않았다면 스토리지 제공자 추가하기, AI 모델 제공자 추가하기, 임베딩 제공자 추가하기 가이드를 따르세요.

단계 1: 새 스토어 추가하기​

스토어는 스토리지, 모델, 임베딩 제공자를 Casibase에 통합하는 데 사용됩니다. 다음 단계에 따라 스토어를 추가할 수 있습니다:

메인 페이지에서 Stores 버튼을 클릭한 다음 Add 버튼을 클릭하여 스토어를 추가합니다.

단계 2: 스토어 세부 정보 입력하기​

이전에 추가한 스토리지 제공자, 모델 제공자, 임베딩 제공자를 선택합니다.

스토어 세부 정보를 입력하고 Save & Exit 버튼을 클릭합니다.

Save & Exit 버튼을 클릭하고 스토어 목록 페이지로 돌아갑니다:

이제 스토어를 사용하여 지식 데이터를 저장하고, 텍스트를 벡터로 변환하고, 챗봇과 대화할 수 있습니다.

다음 섹션에서는 Casibase에서 챗봇과 대화하는 방법을 배워보겠습니다.

멀티 스토어 지원​

멀티 스토어 모드는 각각의 다른 스토어에서 사용자에게 다른 모델과 제안을 제공합니다.

단계 1: 멀티 스토어 활성화하기​

먼저 내장 스토어에서 멀티 스토어 모드를 활성화해야 합니다.

메인 페이지에서 Stores 버튼을 클릭한 다음 store-built-in 버튼을 클릭하여 내장 스토어로 들어갑니다.

아래로 스크롤하여 Can Select Store 필드를 찾아 체크합니다.

단계 2: 사용 가능한 스토어 추가하기​

멀티 스토어 모드는 사용 가능한 스토어만 제공합니다. 스토어를 사용 가능하게 만들려면 해당 스토어의 스토리지 제공자, 모델 제공자, 임베딩 제공자를 구성해야 합니다.

단계 3: 대화를 위한 스토어 선택하기​

Casibase는 스토어를 선택하는 매우 편리한 방법을 제공합니다.

"New Chat" 위에 마우스를 올리면 아래에 나타나는 목록에서 사용하고 싶은 스토어를 선택할 수 있습니다.

"New Chat" 버튼을 클릭하면 기본 스토어가 할당됩니다.
