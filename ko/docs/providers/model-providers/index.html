모델 제공자

소개​

Casibase에 모델 제공자를 추가하면 머신 러닝 모델과 AI 기능을 통합하여 기능을 향상시킬 수 있습니다. 모델 제공자를 통해 지식 베이스 시스템에서 데이터를 분석하고 처리하여 더 스마트하고 효율적으로 만들 수 있습니다.

모델 제공자 추가하기​

모델 제공자는 대규모 언어 모델을 Casibase에 통합하는 데 사용됩니다. 다음 단계에 따라 추가할 수 있습니다:

홈페이지에서 제공자 버튼을 클릭합니다.

모델 제공자를 추가하려면 추가 버튼을 클릭합니다.

모델 제공자 상세 정보 작성​

모델 제공자 상세 정보를 작성하고 저장 및 종료 버튼을 클릭합니다.

팁

Casibase는 다음을 포함한 다양한 모델 제공자를 지원합니다:

  • 허깅 페이스
    • 메타-라마/라마-2-7b
    • THUDM/챗글램2-6b
    • 바이추안-인크/바이추안2-13B-챗
    • 지피티2
    • …
  • 오픈라우터
    • 앤트로픽/클로드-2
    • 팜-2-챗-바이슨
    • 팜-2-코드챗-바이슨
    • 오픈AI/gpt-4
    • …
  • 오픈에이아이
    • 텍스트-다빈치-003
    • gpt-3.5-터보
    • 지피티-4
    • …
주의
  • 카테고리: 모델 제공자의 첫 번째 수준 카테고리입니다. 예를 들어 모델 및 임베딩입니다.
  • 유형: 모델 제공자의 두 번째 수준 카테고리입니다. 예를 들어 OpenAI 및 Hugging Face입니다.
  • 키: OpenAI 계정의 키입니다.

주의

일부 모델은 스트리밍 출력을 지원하지 않습니다. 스트리밍을 지원하는 것으로 알려진 모델은 다음과 같습니다:

  • gpt-3.5-터보-0613

모델 제공자를 추가한 후에는 채팅봇, 질의응답 등의 AI 기능을 통해 Casibase에서 데이터를 분석하고 처리하는 데 사용할 수 있습니다.
