원격 데스크톱 프로토콜

Casibase는 RDP 프로토콜을 통해 노드에 연결하는 것을 지원합니다:

RDP 연결​

  1. Guacamole 서버 시작

    docker 실행 --name guacd -d -p 4822:4822 guacamole/guacd

  2. 새 노드를 추가하고 프로토콜을 rdp로 설정

  3. 연결 버튼을 클릭하여 노드에 연결

원격 애플리케이션​

Windows 노드에서 원격 애플리케이션 사용을 지원합니다. 노드 편집 페이지에서 원격 애플리케이션을 추가한 다음 연결 버튼을 클릭하여 원격 애플리케이션에 연결할 수 있습니다.

  1. 서버 측에서 원격 애플리케이션을 구성합니다.\ RemoteApp Tool을 사용하여 애플리케이션을 등록할 수 있습니다.

  2. 서버 측 구성에 따라 노드 편집 페이지에서 원격 애플리케이션 정보를 구성합니다. 'remoteAppName', 'remoteAppDir' 및 'remoteAppArgs'를 입력해야 합니다. 참조: Guacamole 구성 — Apache Guacamole 매뉴얼 v1.5.3

  3. 원격 애플리케이션에 연결합니다.
