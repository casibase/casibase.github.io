Ajouter un fournisseur de modèle d'IA

Ce document est un tutoriel étape par étape conçu pour les débutants. Il vous guidera à travers le processus d'intégration d'un fournisseur de modèle avec Casibase, notre puissant système de base de connaissances.

Introduction​

L'ajout de fournisseurs de modèles à Casibase améliore ses fonctionnalités en intégrant des modèles d'apprentissage automatique et des capacités d'IA. Les fournisseurs de modèles vous permettent d'analyser et de traiter les données dans le système de base de connaissances, le rendant plus intelligent et efficace.

Si c'est la première fois que vous intégrez un fournisseur de modèle, ne vous inquiétez pas. Nous avons décomposé le processus en étapes simples que n'importe qui peut suivre.

Étape 1 : Déployer Casdoor et Casibase​

Avant d'ajouter un fournisseur de modèle d'IA, assurez-vous que vous avez déjà déployé Casdoor et Casibase. Si vous ne l'avez pas encore fait, consultez le tutoriel Déployer Casdoor et Casibase.

Étape 2 : Ajouter un nouveau fournisseur de modèle​

Les fournisseurs de modèles sont utilisés pour intégrer des LLM à Casibase. Vous pouvez les ajouter en suivant ces étapes :

Cliquez sur le bouton Fournisseurs sur la page d'accueil.

Étape 2.1 : Ajouter un fournisseur de modèle​

Cliquez sur le bouton Ajouter pour ajouter un fournisseur de modèle.

Étape 2.2 : Remplir les détails du fournisseur de modèle​

Remplissez les détails du fournisseur de modèle et cliquez sur le bouton Enregistrer et quitter.

conseil

Casibase prend en charge divers fournisseurs de modèles, notamment :

  • Hugging Face
    • meta-llama/Llama-2-7b
    • THUDM/chatglm2-6b
    • baichuan-inc/Baichuan2-13B-chat
    • gpt2
    • ......
  • OpenRouter
    • anthropic/claude-2
    • palm-2-chat-bison
    • palm-2-codechat-bison
    • openai/gpt-4
    • ......
  • OpenAI
    • text-davinci-003
    • gpt-3.5-turbo
    • gpt-4
    • ......
prudence
  • Catégorie : catégorie de premier niveau du fournisseur de modèle. Par exemple, Modèle et Embedding.
  • Type : catégorie de deuxième niveau du fournisseur de modèle. Par exemple, OpenAI et Hugging Face.
  • CléSecrète : la clé de votre compte OpenAI.

Certains modèles ne prennent pas en charge la sortie en streaming. Les modèles connus pour prendre en charge la sortie en streaming incluent :
