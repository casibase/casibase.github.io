Déployer Casdoor et Casibase

Introduction​

conseil

Qu'est-ce que Casdoor ?

Casdoor est un puissant système d'authentification qui offre une expérience de connexion sécurisée et fiable. C'est un prérequis pour Casibase, alors assurez-vous de le déployer en premier.

Visitez le site Casdoor pour plus d'informations.

Étape 1 : Déployer Casdoor​

Dans le Guide de déploiement de Casdoor, vous pouvez trouver les étapes détaillées pour déployer Casdoor.

Une fois que vous avez déployé Casdoor, vous ressemblerez à ceci :

Étape 2 : Créer une organisation dans Casdoor​

Dans Casdoor, vous pouvez créer une organisation pour gérer vos utilisateurs et applications. Vous pouvez créer une organisation en cliquant sur le bouton Gestion des utilisateurs - Organisations sur la page d'accueil.

Étape 2.1 : Ajouter une organisation​

Cliquez sur le bouton Ajouter pour ajouter une organisation.

Étape 2.2 : Remplir les informations de l'organisation​

Remplissez les informations de l'organisation et cliquez sur le bouton Enregistrer et quitter.

Étape 2.3 : Voir l'organisation​

Après avoir ajouté l'organisation, vous pouvez consulter les informations de l'organisation.

Étape 3 : Créer une application dans Casdoor​

Dans Casdoor, vous pouvez créer une application pour gérer vos utilisateurs et organisations. Vous pouvez créer une application en cliquant sur le bouton Authentification - Applications sur la page d'accueil.

Étape 3.1 : Ajouter une application​

Cliquez sur le bouton Ajouter pour ajouter une application.

Étape 3.2 : Remplir les informations de l'application​

Remplissez les informations de l'application et cliquez sur le bouton Enregistrer et quitter.

Étape 3.3 : Voir l'application​

Après avoir ajouté l'application, vous pouvez consulter les informations de l'application.

Étape 4 : Créer un utilisateur pour Casibase dans Casdoor​

Dans Casdoor, vous pouvez créer un utilisateur pour vous connecter à Casibase. Vous pouvez créer un utilisateur en cliquant sur le bouton Gestion des utilisateurs - Organisations - Utilisateurs sur la page d'accueil.

conseil

Les utilisateurs sont des membres de l'organisation qui peuvent se connecter aux applications de cette organisation.

Visitez le site Casdoor pour plus d'informations.

Étape 4.1 : Ajouter un utilisateur​

Cliquez sur le bouton Ajouter pour ajouter un utilisateur.

Étape 4.2 : Remplir les informations de l'utilisateur​

Remplissez les informations de l'utilisateur et cliquez sur le bouton Enregistrer et quitter.

  • Mot de passe

Vous pouvez définir le mot de passe de l'utilisateur en cliquant sur le bouton Modifier le mot de passe.

  • Administrateur

Vous pouvez définir les droits d'administrateur pour l'utilisateur en cliquant sur le bouton Est-ce un administrateur.

Étape 4.3 : Voir l'utilisateur​

Après avoir ajouté l'utilisateur, vous pouvez consulter les informations de l'utilisateur.

Étape 5 : Déployer Casibase​

Comme pour Casdoor, vous pouvez suivre le Guide de déploiement de Casibase pour déployer Casibase.

Une fois que vous avez déployé Casibase, vous ressemblerez à ceci :
