Bereitstellung von Casdoor und Casibase

Einführung​

Tipp

Was ist Casdoor?

Casdoor ist ein leistungsstarkes Authentifizierungssystem, das eine sichere und zuverlässige Anmeldeerfahrung bietet. Es ist eine Voraussetzung für Casibase, stellen Sie also sicher, dass Sie es zuerst bereitstellen.

Besuchen Sie die Casdoor Website für weitere Informationen.

Schritt 1: Casdoor bereitstellen​

In der Casdoor-Bereitstellungsanleitung finden Sie detaillierte Schritte zur Bereitstellung von Casdoor.

Sobald Sie Casdoor bereitgestellt haben, sieht es folgendermaßen aus:

Schritt 2: Organisation in Casdoor erstellen​

In Casdoor können Sie eine Organisation erstellen, um Ihre Benutzer und Anwendungen zu verwalten. Sie können eine Organisation erstellen, indem Sie auf der Startseite auf die Schaltfläche Benutzerverwaltung - Organisationen klicken.

Schritt 2.1: Organisation hinzufügen​

Klicken Sie auf die Schaltfläche Hinzufügen, um eine Organisation hinzuzufügen.

Schritt 2.2: Organisationsinformationen ausfüllen​

Füllen Sie die Organisationsinformationen aus und klicken Sie auf die Schaltfläche Speichern und Beenden.

Schritt 2.3: Organisation anzeigen​

Nach dem Hinzufügen der Organisation können Sie die Organisationsinformationen anzeigen.

Schritt 3: Anwendung in Casdoor erstellen​

In Casdoor können Sie eine Anwendung erstellen, um Ihre Benutzer und Organisationen zu verwalten. Sie können eine Anwendung erstellen, indem Sie auf der Startseite auf die Schaltfläche Authentifizierung - Anwendungen klicken.

Schritt 3.1: Anwendung hinzufügen​

Klicken Sie auf die Schaltfläche Hinzufügen, um eine Anwendung hinzuzufügen.

Schritt 3.2: Anwendungsinformationen ausfüllen​

Füllen Sie die Anwendungsinformationen aus und klicken Sie auf die Schaltfläche Speichern und Beenden.

Schritt 3.3: Anwendung anzeigen​

Nach dem Hinzufügen der Anwendung können Sie die Anwendungsinformationen anzeigen.

Schritt 4: Benutzer für Casibase in Casdoor erstellen​

In Casdoor können Sie einen Benutzer erstellen, um sich bei Casibase anzumelden. Sie können einen Benutzer erstellen, indem Sie auf der Startseite auf die Schaltfläche Benutzerverwaltung - Organisationen - Benutzer klicken.

Tipp

Benutzer sind Mitglieder einer Organisation und können sich bei den Anwendungen dieser Organisation anmelden.

Besuchen Sie die Casdoor Website für weitere Informationen.

Schritt 4.1: Benutzer hinzufügen​

Klicken Sie auf die Schaltfläche Hinzufügen, um einen Benutzer hinzuzufügen.

Schritt 4.2: Benutzerinformationen ausfüllen​

Füllen Sie die Benutzerinformationen aus und klicken Sie auf die Schaltfläche Speichern und Beenden.

  • Passwort

Sie können das Benutzerpasswort festlegen, indem Sie auf die Schaltfläche Passwort ändern klicken.

  • Administrator

Sie können die Administratorrechte des Benutzers festlegen, indem Sie auf die Schaltfläche Ist Administrator klicken.

Schritt 4.3: Benutzer anzeigen​

Nach dem Hinzufügen des Benutzers können Sie die Benutzerinformationen anzeigen.

Schritt 5: Casibase bereitstellen​

Ähnlich wie bei Casdoor können Sie der Casibase-Bereitstellungsanleitung folgen, um Casibase bereitzustellen.

Sobald Sie Casibase bereitgestellt haben, sieht es folgerleicht aus:
